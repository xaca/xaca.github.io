Creación de una aplicación web

En este tutorial, podrás seguir los pasos necesarios para crear el frontend de una aplicación, es decir la interfaz que usará el usuario para iteractuar con una plataforma. Luego conectarás el backend, es decir todo lo que sucede detrás de una aplicación para que sea posible almacenar, consultar y realizar diferentes operaciones con los datos.

Comencemos

  1. Ingresar a codepen, este es un editor en línea, esta herramienta nos permite crear la aplicación y trabajar desde cualquier computador que tenga un navegador y una conexión a internet.
  2. La interfaz gráfica de nuestra aplicación esta compuesta por 2 secciones: La estructura que se define por medio del html y el look gráfico, el cual es un conjunto de reglas visuales que permiten personalizar las propiedades que definen el estilo de la interfaz del usuario.
  3. Hay una tercera sección, denominada javascript, se encarga de la interacción del usuario con la aplicación, por medio de programación. Nos permitirá, agregar respuestas a las acciones del usuario, por ejemplo cuando presiona un botón.

    Si presionamos el botón actualizar, debe aparecer una ventana, como vemos a continuación

  9. Ahora vamos a automatizar el proceso que hemos hecho hasta el momento, modifica el siguiente código en la sección de javascript para que mires el resultado

                        
function cargarDatos()
{
    let aleatorio = Math.round(Math.random()*(100-1)+parseInt(1));;
    let url = "https://pokeapi.co/api/v2/pokemon/"+aleatorio;
    fetch(url)
    .then(function(response) {
    return response.json();
    })
    .then(function(json) {
    mascota.src = json.sprites.front_default;
    nombre.innerHTML = json.name;
    peso.innerHTML = json.weight;
    });
}
