Creación de una aplicación web

En este tutorial, podrás seguir los pasos necesarios para crear el frontend de una aplicación, es decir la interfaz que usará el usuario para iteractuar con una plataforma. Luego conectarás el backend, es decir todo lo que sucede detrás de una aplicación para que sea posible almacenar, consultar y realizar diferentes operaciones con los datos.

Comencemos

Ingresar a codepen, este es un editor en línea, esta herramienta nos permite crear la aplicación y trabajar desde cualquier computador que tenga un navegador y una conexión a internet.

La interfaz gráfica de nuestra aplicación esta compuesta por 2 secciones: La estructura que se define por medio del HTML y el look gráfico CSS, el cual es un conjunto de reglas visuales que permiten personalizar las propiedades que definen el estilo de la interfaz del usuario.

Hay una tercera sección, denominada JS (javascript), se encarga de la interacción del usuario con la aplicación, por medio de programación. Nos permitirá, agregar respuestas a las acciones del usuario, por ejemplo cuando presiona un botón.
